# AICODER 的 vip1 全栈教程项目源码

## 如何使用

下载项目文件夹

```sh
$ git clone git@github.com:malun666/aicoder_vip1_code.git
```

全局安装 json-server

```sh
$ npm i -g json-server
```

启动 json-server 的服务

```sh
# 安装nodemon
$ npm i -g nodemon

# 启动服务
$ nodemon server.js
```
